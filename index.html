About Dark Academia

A gallery built for exploration

Dark Academia is a curated photography showcase featuring nature, architecture, people, and abstract imagery. Browse the grid, filter by category, and open any photo in a fullscreen lightbox with keyboard navigation.

This project demonstrates responsive layout, smooth CSS transitions, and vanilla JavaScript—no frameworks required. Perfect for portfolios, internships, and learning modern front-end basics.
